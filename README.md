# NutriTrack - Calorie Tracking REST API

## What It Does
NutriTrack is a lightweight, layered REST API designed to help users monitor their daily nutritional intake, manage food items and meals, and analyze eating habits through data aggregation.

## Tech Stack
- Python 3.x
- FastAPI / Flask
- SQLite (Development) & MySQL (Production)
- Pandas (Analytics)
- Unittest
